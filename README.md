## Dataset

Data Science Careers and Salaries 2025, publicado en Kaggle (944 registros,
13 columnas)

Link: https://www.kaggle.com/datasets/nalisha/data-science-careers-and-salaries-2025/data

## Cómo ejecutar

1. Abre el notebook de la carpeta notebook/ en Google Colab.
2. Ejecuta todas las celdas en orden, de arriba hacia abajo (menú Entorno de ejecución → Ejecutar todas).
3. Durante la ejecución, acepta los permisos cuando Colab los solicite:
  - acceso a Google Drive, para guardar algunas figuras del proyecto.
4. El notebook descarga el dataset desde Kaggle de forma automática y reproduce todos los resultados, tablas y figuras incluidos en el informe.

Importante: las celdas deben correrse en orden, ya que cada sección usa
variables creadas en la anterior. Si se ejecuta una celda fuera de orden puede
aparecer un error de variable no definida.
